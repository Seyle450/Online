4. Reichweitenmessung und Einwilligung

Diese Website bindet ein Analyse-Skript des Seitenbetreibers ein, das die Nutzung anonymisiert erfasst (z. B. welche Inhalte aufgerufen werden, woher Besucher kommen, was geklickt wird) — ohne Namen oder Kontaktdaten, ohne Verkauf oder Weitergabe an Dritte. Beim ersten Besuch werden Sie in einem Einwilligungsfenster gefragt, ob Sie dieser Messung zustimmen möchten. Ihre Wahl wird lokal in Ihrem Browser gespeichert (Local Storage bzw. Cookie), damit Sie nicht bei jedem Besuch erneut gefragt werden; Sie können diese Wahl jederzeit über die Einstellungen Ihres Browsers zurücksetzen. Weitere Informationen zu diesem Dienst finden Sie in der Datenschutzerklärung des Anbieters unter elyesferchichi.com/datenschutz.html.

6. Schriftarten (lokal eingebunden)

Diese Website nutzt zur einheitlichen Darstellung Schriftarten, die lokal auf unserem eigenen Server gespeichert sind. Beim Aufruf der Seite wird dafür keine Verbindung zu Servern Dritter (z. B. Google) hergestellt; es werden keine Daten an Dritte übertragen.

7. Google Maps (Zwei-Klick-Einbindung)

Auf dieser Website kann eine Karte des Anbieters Google Maps (Google Ireland Limited) angezeigt werden, um unseren Standort darzustellen. Die Karte wird erst nach Ihrem ausdrücklichen Klick auf „Karte laden" eingebunden — erst dann wird eine Verbindung zu Servern von Google hergestellt, wobei auch Ihre IP-Adresse übertragen werden kann. Ohne diesen Klick werden keine Daten an Google übermittelt. Weitere Informationen zum Umgang mit Nutzerdaten finden Sie in der Datenschutzerklärung von Google unter policies.google.com/privacy.
